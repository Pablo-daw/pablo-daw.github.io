Ejemplo de iconos de Bootstrap

¿Cómo usar los iconos de Bootstrap?

Paso 1: Incluya el siguiente enlace de Bootstrap en la sección <head> de su documento HTML:
<link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/3.3.7/css/bootstrap.min.css"> para la version 3 y para la última versión: <link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/bootstrap-icons@1.5.0/font/bootstrap-icons.css">

Paso 2: Para colocar íconos en el <body> en su página web, use la sintaxis <i class="glyphicon glyphicon-*"></i>(versión 3) o <i class="bi bi-*"></i>(última versión), donde * representa el nombre del ícono en particular.
Por lo tanto, si un icono es una nube, el código HTML del icono sería <i class="glyphicon glyphicon-cloud"></i>(versión 3) o <i class="bi bi-cloud"></i>(última versión), de manera similar para el resto de iconos.
